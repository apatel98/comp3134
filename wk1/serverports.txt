• What is the definition of server port?
- At the software level, within an operating system, a port is a logical 
construct that identifies a specific process or a type of network service.
Ports are identified for each protocol and address combination by 16-bit 
unsigned numbers, commonly known as the port number.

• What is server port range?
- 0 to 65535

• How many bits does number above represent?
- 16 bit
